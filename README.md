# Glip6

**One Platform. Infinite Possibilities.**

Glip6 is the digital hometown for the next generation. We are replacing 5-7 disconnected apps with one seamless experience that brings your work, social life, and automation together.

## Features

-   **Unified Workspace**: Seamlessly integrate tasks, docs, and AI tools.
-   **Social Hub**: Real-time chat, voice spaces, and community.
-   **Smart Automation**: Automate workflows with intelligent controls.

## Tech Stack

-   **Framework**: Next.js 15 (App Router)
-   **Styling**: Tailwind CSS v4
-   **Animations**: Framer Motion
-   **3D Elements**: Three.js / React Three Fiber

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
